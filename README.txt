Pour installer:

Extraire le dossier

Lancer Mamp
Dans MAMP/Preference/Web Server, placer le chemin vers le dossier décompresser.

Aller ensuite dans phpMyAdmin, céer une base de donnée du nom de projetweb puis
importer le fichier projetweb.sql à l'intérieur.

Ensuite lancer le site en tappant localhost/index.php dans la barre d'URL
